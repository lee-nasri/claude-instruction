# order-consumer

> This is an example knowledge file for **order-consumer**.
> In practice, this file contains the service overview, queue subscription setup,
> order matching logic, settlement flow, downstream event publishing, and error
> handling patterns that Claude needs to understand before working on this service.
>
> **How to generate this for your own service:**
> 1. Open Claude Code inside your project directory
> 2. Run `/init` — Claude will read your codebase and generate a `CLAUDE.md`
> 3. Ask Claude: *"Summarize this into a knowledge file for ~/.claude/knowledge/"*
> 4. Save the output as `~/.claude/knowledge/order-consumer.md`
