# notification-api

> This is an example knowledge file for **notification-api**.
> In practice, this file contains the service overview, supported channels
> (push, email, SMS), notification templates, retry strategy, and provider
> integrations that Claude needs to understand before working on this service.
>
> **How to generate this for your own service:**
> 1. Open Claude Code inside your project directory
> 2. Run `/init` — Claude will read your codebase and generate a `CLAUDE.md`
> 3. Ask Claude: *"Summarize this into a knowledge file for ~/.claude/knowledge/"*
> 4. Save the output as `~/.claude/knowledge/notification-api.md`
